A portable devboard designed to be a small customizable gadget. It includes 3 Buttons, a 0.91 inch OLED screen and is power by an ESPRESSIF ESP32-WROOM-32E-N4. 
Power is from a usb port.

This project was originally meant to be a mini mp4 player, but it can also be customized, the buttons used for control and the small screen for display. 

Other Features:
- Power indicator LEDS
- Boot/RST Buttons
- USB-C Powered
- Micro SD Card slot for audio, image, data storage.
